# Day 05 - OOP Mastery Challenge 1

## Project
University Management System

## Concepts Practiced
- Encapsulation
- Inheritance
- Abstraction
- Runtime Polymorphism
- Method Overriding
- Static Members

## Class Hierarchy

Person (Abstract)
├── Student
├── Professor
└── Staff

## Features
- Manage university members
- Calculate income dynamically
- Store different objects in Person[]
- Runtime polymorphism using parent references
- Count total persons created

## Key Learning
- Parent references can store child objects
- Method calls are resolved at runtime
- Abstract classes provide a common contract
- Encapsulation protects object state
